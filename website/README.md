# Website Content – ICAN

This folder contains **all draft content, page structure, and assets** for the public website IowaCannabisAction.org.

## Purpose
To provide a workspace for:
- Page content drafts  
- Copywriting  
- Branding assets  
- Sitemap planning  
- Program descriptions  
- Public education pages  

## Subfolders
- `pages/` – Draft versions of site pages  
- `assets/` – Images, graphics, icons  
- `drafts/` – Early content not ready for publication  
- `sitemap.md` – Site architecture & navigation  

## Notes
When finalized, website assets will be published via GitHub Pages or Softr.
