# ICAN Public Website Source

This directory houses the source material for the Iowa Cannabis Action Network (ICAN) public website. As a 501(c)(4) social welfare organization, ICAN uses the site to inform members, supporters, and the public about advocacy priorities, campaigns, and ways to participate.

Content that belongs here includes:
- Static pages, blog posts, and educational materials approved for public release.
- Web assets such as images, stylesheets, and scripts that support the site experience.
- Configuration or build files for the chosen web framework or static site generator.

Do **not** store internal documents or confidential information here. All files in this folder should be suitable for public distribution and aligned with ICAN's mission and compliance obligations.
