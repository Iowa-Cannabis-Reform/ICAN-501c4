# Operations – ICAN

This folder contains the **internal operational framework** for ICAN, including skills training, volunteer systems, program management, and policy organizing.

## Purpose
To store:
- Program manuals  
- Volunteer workflow documents  
- Community outreach materials  
- Internal scripts  
- Administrative operations  
- Victory Garden infrastructure

## Subfolders
- `Victory-Gardens/` – Full program architecture  
- `volunteer/` – Onboarding, training, handbook  
- `policy/` – Issue briefs, research, advocacy tools  
- `community/` – Events, partnerships, local outreach  

## Notes
PAC/527 operations are not permitted here.
