# Victory Gardens – ICAN

This folder houses all internal work related to ICAN’s flagship **Victory Gardens Program**, a community initiative focused on gardening, regenerative practices, food bank contributions, and grassroots advocacy.

## Subfolders
- `marketing/` – Social assets, announcements, promotional materials  
- `resources/` – PDF toolkits, signage, garden support materials  
- `scripts/` – Event scripts, onboarding scripts, outreach scripts  

## Core Files
- `architecture.md`
- `onboarding.md`
