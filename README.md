# Iowa Cannabis Action Network (ICAN) – 501(c)(4)
This repository contains all internal materials for **Iowa Cannabis Action Network, Inc.**, a 501(c)(4) social welfare organization dedicated to cannabis education, advocacy, public engagement, and community-based programs such as Victory Gardens.

This repo is the **single source of truth** for:
- Governance & legal compliance  
- Advocacy and educational materials  
- Operational systems  
- Program work (Victory Gardens, outreach, research)  
- Communications and branding  
- Website content drafts  
- Internal SOPs and policies  

ICAN’s mission is to advance public understanding of cannabis, strengthen community capacity, promote equitable policy, and build a statewide network for reform.
