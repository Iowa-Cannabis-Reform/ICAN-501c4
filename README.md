    # Iowa Cannabis Action Network (ICAN) – 501(c)(4) Social Welfare Organization

    This repository contains **all internal materials** for:

    **Iowa Cannabis Action Network (ICAN)**  
    Type: 501(c)(4) Social Welfare Organization  
    Domain: IowaCannabisAction.org

    ## Purpose

    This repo is the **single source of truth** for this legal entity. It should contain everything that *only* this entity controls or uses.

    **Primary focus areas:**
    - Cannabis advocacy and public education
- Issue-based organizing and coalition building
- Community outreach and nonpartisan policy awareness

    ## Folder Structure

    - `legal/` – bylaws, policies, filings, compliance drafts
    - `communications/` – messaging specific to this entity
    - `operations/` – internal systems, SOPs, planning docs
    - `website/` – files for the public-facing website (or content feeding into Softr/other CMS)

    ### Special Notes for 501(c)(4)
- Avoid direct candidate coordination in this repo.
- Focus on issues, education, and advocacy messaging.
- Keep any PAC-related discussions inside the PAC repo.

    ## Notes

    - Keep strategy, donor information, and compliance documents **inside this repo only**.
    - Avoid sharing political strategy files between entities to preserve legal firewalls.
    - Use the `Shared-Infrastructure` repo only for neutral, non-political, reusable assets.
